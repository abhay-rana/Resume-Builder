## [1.0.0] - 2025-01-11

### Added

#### Core Features
- **Resume Builder** - Full-featured resume editor with real-time preview
- **PDF Export** - High-quality PDF generation with exact page breaks
- **Local Storage** - Resume data persisted in browser (no account needed)

#### Templates
- **Modern & Clean** - Contemporary single-column design
- **Two Column** - Efficient space-maximizing layout
- **Executive Style** - Traditional professional template
- **Creative Modern** - Distinctive design for creative roles

#### AI Features
- **PDF Import** - Upload existing resume, AI extracts all information
- **LinkedIn Import** - Import profile data with one click
- **Smart Parsing** - Intelligent section detection and formatting

#### Customization
- **Color Options** - Full color customization for all templates
- **Font Selection** - 20+ professional fonts
- **Spacing Controls** - Margins, line height, section spacing
- **Icon Options** - Toggle icons on/off

#### Resume Sections
- Personal Details
- Professional Summary
- Work Experience
- Education
- Skills
- Projects
- Certifications
- Languages
- Custom Sections

#### Design Features
- Real-time PDF preview
- Drag-and-drop section reordering
- Rich text editing
- Responsive design
- Print-optimized output

### Technical
- React 18 with TypeScript
- Redux Toolkit for state management
- Tailwind CSS for styling
- Vite for fast development
- ATS-optimized HTML output
