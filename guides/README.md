## Quick Tips

### Do's
- Use action verbs to start bullet points
- Quantify achievements with numbers
- Tailor resume to each job application
- Keep formatting clean and consistent
- Save as PDF for consistent formatting

### Don'ts
- Use personal pronouns (I, me, my)
- Include irrelevant experience
- Use generic objective statements
- Add photos (in US/UK)
- List job duties instead of achievements
